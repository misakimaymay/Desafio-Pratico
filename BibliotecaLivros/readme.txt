# BibliotecaLivros

Este projeto é um sistema simples de gerenciamento de uma biblioteca de livros, desenvolvido em C# (.NET 8). Ele permite cadastrar, listar, buscar, atualizar e remover livros utilizando um menu interativo no console.

## Funcionalidades

- **Cadastrar Livro:** Adiciona um novo livro informando título, autor e ISBN.
- **Listar Livros:** Exibe todos os livros cadastrados.
- **Buscar Livros:** Permite buscar livros pelo título ou autor.
- **Atualizar Livro:** Atualiza o título e autor de um livro a partir do ISBN.
- **Remover Livro:** Remove um livro pelo ISBN.

## Como usar

1. Compile o projeto com o .NET 8 SDK.
2. Execute o programa.
3. Utilize o menu exibido no console para interagir com a biblioteca.

## Estrutura do Código

- `Livro`: Classe que representa um livro.
- `Biblioteca`: Classe que gerencia a lista de livros e as operações.
- `Program`: Classe principal com o menu int
